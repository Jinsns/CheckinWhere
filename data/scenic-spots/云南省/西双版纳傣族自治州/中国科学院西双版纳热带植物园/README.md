# 中国科学院西双版纳热带植物园

## 景区概览

坐标：21°55′39″N 101°15′15″E / 21.9276°N 101.2542°E / 21.9276; 101.2542 维基百科，自由的百科全书 中国科学院西双版纳热带植物园 中国科学院西双版纳热带植物园（英語：XiShuangBanNa Tropical Botanical Garden, Chinese Academy of Sciences），位于云南省西双版纳傣族自治州勐腊县勐仑镇葫芦岛，从景洪市市区出发需乘车到达，為中华人民共和国国家5A级旅游景区，面积约900公顷。
