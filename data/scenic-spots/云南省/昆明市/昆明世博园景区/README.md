# 昆明世博园景区

## 景区概览

坐标：25°04′49″N 102°45′55″E / 25.0804°N 102.7653°E / 25.0804; 102.7653 维基百科，自由的百科全书 世界园艺博览园简称世博园，是1999年昆明世界园艺博览会的主会址，位于云南省昆明市盘龙区青云街道，与金殿公园依山相伴，植被覆盖率约80%，其中有120公顷灌木丛茂密的缓坡，水面占10％～15％。 世博园占地约220公顷，由5个场馆（国际馆、中国馆、人与自然馆、科技馆、大温室）及３个大室外展区（32个国外展区、34个国内展区、7个专题园区）组成。其中，专题园区有：蔬菜瓜果园，竹园，盆景园，茶园，药草园，树木园和名花异石园。
