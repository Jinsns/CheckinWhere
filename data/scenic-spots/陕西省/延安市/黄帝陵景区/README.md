## 🎯 游览实用指南

### 🚗 交通指南

黄帝陵在陕西延安黄陵县，离西安大约180公里。

**从西安出发**：
- **高铁**：西安北站→黄陵南站，约1小时，二等座50元，然后打车到景区，约10分钟
- **大巴**：西安城北客运站→黄陵，约2.5小时，50元
- **自驾**：包茂高速，约2小时，过路费约90元
- **景区内交通**：景区不大，全程步行即可

### 🎫 门票信息（2025年参考）
- **门票**：旺季75元（3-11月），淡季50元（12-2月）
- **半价票**：学生、60-64岁老人
- **免票**：65岁以上、军人、残疾人、记者
- **讲解**：官方讲解100元/次，建议请
- **预约**：关注"黄帝陵景区"公众号预约
