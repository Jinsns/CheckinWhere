# 天山托木尔景区

## 景区概览

坐标：42°02′05″N 80°07′43″E / 42.03466°N 80.12871°E / 42.03466; 80.12871 维基百科，自由的百科全书 托木尔峰（维吾尔老文字：تۆمۈر چوققىسى，拉丁维吾尔文：Tömür choqqisi，吉爾吉斯語：Жеңиш чокусу，羅馬化：Jeŋiş çokusu，俄語：Пик победы，羅馬化：Pik pobedy，哈薩克語：Жеңіс шыңы，羅馬化：Jeñis şıñı），是天山山脉最高峰。也為吉爾吉斯之全國最高峰。苏联出版的地图上标明海拔7439米，1比100万《中华人民共和国地图》标高則為海拔7443米。
