# 白沙湖景区

## 景区概览

白沙湖[位于中国新疆的国家AAAAA级旅游景区]_百科 白沙湖位于中国新疆维吾尔自治区哈巴河县、中国-哈萨克斯坦边境的185团境内，距离哈巴河县城108千米。该湖南北长800米，东西宽600米，形状为东凹西凸，是一个位于沙山之中的小湖。湖水清澈、动植物种类丰富，与周边鸣沙山形成对比，被称为“塞北小江南”。该景区景点包括白沙湖、鸣沙山、眼睛山、185团抗洪守土纪念碑、西北边境第一连等自然及人文景点。 2017年2月，白沙湖景区正式列入国家AAAAA级旅游景区。2019年，白沙湖所在的185团入选国家首批全域旅游示范区。 自然地理 位置境域 白沙湖景区位于哈巴河县185团境内，距离县城108千米，是中国第二大沙漠古尔班通古特沙漠北面的一个淡水湖。此外，新疆境内还有一座白沙湖位于南疆帕尔高原上的克州阿克陶县境内、喀喇昆仑山脉中，海拔约3300米。 气候 白沙湖景区属于温带大陆性气候，冬季严寒漫长，风大多雪;夏季炎热高温。该地年平均气温4摄氏度，极端高温39摄氏度(1974年7月11日)，极端低温零下50摄氏度(1969年1月27日);年平均降水量178毫米。当地盛行刮东风和西风，年平均八级以上大风42天;一次刮风持续时间为7天，一般3天东风3天西风。 地形地貌 白沙湖湖区面积0.5平方千米;海拔650米;形状为东凹西凸，南北长800米、东西宽600米。除湖本身外，景区以沙地为主要地貌，沙化土地面积约124平方千米。沙地类型被分为天然固定沙地、天然半固定沙地和流动沙地3种。 生物多样性 白沙湖里生长着荷花，四周生长有芦苇、芨芨草、长苞香蒲、拂子茅、水莎草、球穗藨草、细秆藨草、水葱、沙苔草等湿地... 高質 帕米爾高原白沙湖攻略：解鎖沙山之謎與藍白之幻 帕米爾高原的白沙湖,是南疆之行的必打卡秘境,這裡藏著「沙山成因之謎」的地質奇觀,更有白沙映藍湖、雪山襯碧空的「藍白之幻」,這份實用攻
