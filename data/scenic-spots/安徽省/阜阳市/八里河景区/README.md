# 八里河景区

## 景区概览

八里河旅游区位于安徽省阜阳市东南的颍上县，南临淮水，东濒颍河，北距颍城8千米，西迄阜阳60千米，东南距合肥170千米，总占地15.8平方千米。 八里河旅游区属于内陆湖，处于暖温带向亚热带过渡带，四季分明，常年平均降水量为923.8毫米，年平均气温15.1摄氏度，年平均日照时数达2213.3小时。海拔18.5-21.5米，正常水面1000公顷，雨量充沛。湿地类型丰富，有80种浮游植物和70余种浮游动物，以及国家一级保护动物东方白鹳、白头鹤等八里河景区由农民建造而来，分为鸟语林、西区、东区、十二花岛、明清苑五大景点。西区和东区分别有世界著名建筑和东方建筑的仿制景观，别具特色。 1993年，八里河被联合国授予环保“全球五百佳”称号。2004年八里河镇成为全国首批“全国农业旅游示范点。2013年10月11日，八里河旅游区被国家旅游局授予“国家AAAAA级风景区”称号。2014年11月3日，八里河风景区正式成为国家级水利风景区。 自然地理 位置境域 八里河旅游区位于中国安徽省阜阳市颍上县南部的八里河镇，总占地15.8平方千米。南临淮河，东接颍河，向北和向西分别距约颍城县8千米、阜阳市60千米，东南距合肥市170千米。八里河旅游区因其良好的自然地理环境被安徽省列为省级自然保护区。保护区位于颍河和淮河的交汇处，地跨本县八个乡镇，位于东经116°1′-116°40′E，北纬32°27′-32°54′N之间。保护区有“一河两湾五湖”之称，它以八里河为中心，周围有张家湾、马家湾、戴家湖、邱家湖、唐垛湖、第三湖和五里湖。 气候 八里河在暖温带向亚热带过渡带，常年平均降水量为9... 八里河网红玻璃天桥 八里河悬空水上观光玻璃天桥，全长425米，宽3米，全高55米，采用全透明特种玻璃建造，像一条蜿蜒的飞龙盘旋在八里河水面之上。同时将9D声光电碎彩屏、花鸟游鱼立体特效等流行科技元素与桥面结合震动..
